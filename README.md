# 📈 Gradient Boosting Regression - Ad Spend Prediction

A machine learning project that uses **Gradient Boosting Regression** to predict sales based on advertising spend across different marketing channels.

## 🚀 Features

- Data preprocessing and analysis
- Gradient Boosting Regressor implementation
- Model training and evaluation
- Sales prediction based on ad spend
- Performance metrics visualization

## 🛠 Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## ▶️ Run the Project

```bash
git clone https://github.com/abhinav29890/GradientBoosting_Regression_Adspend.git

cd GradientBoosting_Regression_Adspend

python main.py
```
